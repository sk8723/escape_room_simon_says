The code is in a perception > planning > action format with finite state machines to enable readability.

There are several primary functions associated with the user at the start of the game:
Easy Mode: Hold the red button and the yellow button for 3 seconds to start a game in easy mode. The pattern will not contain any sounds to memorize, only flashing lights. Each successive round will speed up. This game mode is also for users who are hard of hearing.
Hard Mode: Hold the red button and the blue button for 3 seconds to start a game in hard mode. Each round will have a new kind of pattern to memorize. The first round will be flashing lights, the second will be animal sounds, and the third will be notes from a steel drum.
Manual Reset: If at any point the game becomes too difficult, the user may start over by holding the red and blue buttons for 3 seconds. They may then select the difficulty mode by holding down the appropriate buttons.
Easter Egg: At the start of the game, there is an Easter egg in reference to the Konami code. If a user presses the following sequence
RED > RED > YELLOW > YELLOW > GREEN > BLUE > GREEN > BLUE > RED > BLUE
then the speaker will start playing the sea shanty What Do You Do With A Drunken Sailor? The user may stop the song at any point and start a game by holding down the appropriate buttons.
